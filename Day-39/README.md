# Day 39 Overview

## Introduction to Functions

These are reusable blocks of code that perform a specific task, which can be executed whenever needed.

Functions can accept inputs called parameters and the actual values passed to the function are called arguements.

Functions in JS create their own scope, variables that are declared inside a function are local to that function and cannot be accessed outside of it.

### Function Declaration

It a defines a named function that can be invoked anywhere in the scope it was declared.

### Function Expression

It is used to define a function expression as a part of a larger expression such as a variable assignment.

### Arrow Functions

It provides a concise syntax for writing function expressions.
