# Day 51 Overview

## Introduction to Backend, Node.js and NPM

### Node.js

- Node.js is an Open-Source,JS runtime environment that allows you to run JS on the server side.

- It is designed for the purpose of building scalable network applications, mainly those that require real-time communication.

- Node.js is built on chrome's V8 JS engine, which makes it fast and efficient.

### Key Features of Node.js

1. Single Threaded - Node.js operates on a single thread, it uses async I/O to handle multiple operations concurrently. This makes it efficient for I/O bound tasks.

2. NPM - Node.js comes with NPM, which is considered as the largest ecosystem of open-source libraries in the world.

3. Cross-Platform - Node.js can run on various OS like Windows, Linux and macOS.

4. V8 Engine - Node.js is built on the V8 engine, which is used to compile JS to machine code, making it faster and efficient.

### NPM

- NPM is a package manager for JS and it is the default package manage for Node.js.

- It helps the developers to share and reuse the code by managing the dependencies.

- Initializing a project using NPM 

    - npm init 
    - npm install package-name

### Key Features of NPM 

1. Package Management - It allows you to manage the dependencies that are present in the project through a 'package.json' file, which mainly lists the packages your project depends on. 

2. Version Control - It helps in managing the different versions of packages to avoid the compatibility issues.
