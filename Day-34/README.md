# Day 34 Overview

## CSS Filters

CSS filters allows you to apply graphical effects like blurring, sharpening, or color shifting to elements, similar to filters in image editing softwares.

The 'filter' property is used to define filter effects on elements.

Multiple filters can be applied by listing them space-seperated within a single 'filter' property.

### CSS Filter Functions

1. blur()

   - Applies a Gaussian blur to the element.

   - The value defines the radius of the blur in pixels.

2. brightness()

   - Adjusts the brigthness of the element.

   - The value is a percentage or a number.

3. contrast()

   - Adjusts the contrast of the element.

   - The value is a percentage or a number.

4. grayscale()

   - Converts the element to grayscale.

   - The value is a percentage.

5. invert()

   - It is used to invert the color of the element.

   - The value is a percentage.

6. opacity()

   - It is used to adjust the opacity of the element.

   - The value is a percentage or a number.

7. saturate()

   - It is used to adjust the color saturation of the element.

   - The value is a percentage or a number.
