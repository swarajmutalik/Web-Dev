# Structure of an HTML Website 

## !DOCTYPE html

This declaration tells the browser that the document is written in HTML5. It is required at the beginning of an HTML document to ensure proper rendering and parsing.

## html lang="en" 

This tag indicates the start of the HTML document. The lang attribute specifies the language of the document, in this case, English (en). It should be set to the appropriate language code for the content of the document.

## head tag

This section contains meta-information about the document, such as its title, character encoding, and viewport settings. It does not display any content directly on the web page.

## meta charset="UTF-8"

Specifies the character encoding for the document. UTF-8 is a widely-used encoding that supports a wide range of characters from various languages.

## meta name="viewport" content="width=device-width, initial-scale=1.0"

Configures the viewport for the webpage. It ensures that the width of the page is set to the width of the device and the initial zoom level is set to 1.0, which means the page is not zoomed in or out initially.

## title tag

Sets the title of the webpage, which appears in the browser's title bar or tab.

## link rel="stylesheet" href="style.css"

Links an external stylesheet (style.css) to the HTML document. CSS (Cascading Style Sheets) files define the visual appearance and layout of the HTML content.

## body tag
This section contains the main content of the webpage that will be displayed to the user.


## script tag
Links an external JavaScript file (script.js) to the HTML document. JavaScript files contain code that adds interactivity and functionality to the webpage.
