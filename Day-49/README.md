## Classes and objects in JS

- Object Literals are a simple way of creating objects using a pair of curly braces. Properties and methods are defined inside the braces.

- Prototypes are a mechanism by which the JS objects inherit features from one another.

  - **proto** property is a reference to the prototype object from which the object inherits the properties and methods.

- Classes in JS provide a clean and a straightforward syntax to create objects and handle inheritance.

- Constructor method is a special method for creating and intializing objects created with a class.

  - When a new instance of the class is created, then
    the constructor is called.

- Inheritance is a feature that allows a class to inherit properties and methods from another classs using the 'extends' keyword. 

    - 'super' keyword is used to call the constructor of the parent class and to access properties and methods of the parent class. 

    - This is a necessary step when you want to initialize the parent class with some values from the child class. 

- Method Overriding allows a subclass to provide a specific implementation of a method that is already defined in its superclass. 

- Instantiation is the process of creating an instance from a class. The 'new' keyword is used to create an instance of a class.
