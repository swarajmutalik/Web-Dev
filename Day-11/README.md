# Day 11 Overview

## Semantic tags

Semantic tags in HTML are elements that have meaningful names that describe their content or role in a webpage. 

They provide a clearer understanding of the document's structure, helping developers, search engines better interpret the content. 

Semantic tags improve the accessibility, readability, and SEO (Search Engine Optimization) of web pages.

### Why Semantic Tags?
Semantic tags are essential due to the following reasons - 

- Convey meaning: The names of these tags inherently suggest their purpose.

- Improve SEO: Search engines use these tags to understand and rank content more accurately.

- Aid accessibility: Assistive technologies rely on semantic tags to navigate and interact with the content.

- Promote maintainability: They create a clear structure, making code easier to read, understand, and maintain.


Here are some commonly used semantic tags in HTML - 

- header : Represents the header section of a webpage or a section within a page. It can contain navigation links, logos, or introductory content.

- nav : Represents a navigation section, typically containing links to different parts of the website.

- section : Represents a thematic grouping of content, usually with a heading.

- article : Represents an independent piece of content that could stand alone, such as a blog post, news article, or forum post.

- aside : Represents content that is tangentially related to the main content, like sidebars or callout boxes.

- footer : Represents the footer section of a webpage or a section, typically containing information like copyright, contact details, or site navigation links.

- main : Represents the main content of a webpage, excluding headers, footers, and sidebars.

- figure : Represents self-contained content, often with a caption. Commonly used with images, diagrams, or code examples.

- figcaption : Represents the caption for the content within a figure tag.

- details : Represents a collapsible section of content.

- summary : Represents a summary or label for a collapsible section.
