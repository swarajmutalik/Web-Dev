# Day 23 Overview

## CSS Overflow

The CSS Overflow property controls what happens when content overflows its containing box. It is essential for managing the display of content that exceeds the bounds of its parent element, particularly when dealing with fixed-size containers or responsive layouts.

### Key Values of Overflow property

1. hidden - content that overflows the container is clipped and no scrollbars are provided.

2. scroll - It always provides a scrollbar, even when the content it not overflowing.

3. auto - scrollbars appear only when the content overflows.

4. overflow-x - It is used to control the horizontal overflow.

5. overflow-y - It is used to control the vertical overflow.
