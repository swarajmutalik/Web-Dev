## CSS Float and Clear

These CSS properties are used to control the flow of elements within a webpage, affecting the layout and how elements interact with each other.

### CSS Float

This property is used to position an element to the left or right of its containing block allowing other elements to flow around it.

Some common values for CSS float are - 

- left - The element floats to the left side of its containing block. 

- right - The element floats to the right side.

- none - It means the element does not float and stays in the normal document flow. 

### CSS Clear 

This property is used to prevent elements from flowing around floated elements. 

It is typically applied to an element that should be below a floated elemen, ensuring it starts on a new line, clearing the effects of previous floats. 

Some common values for CSS clear are -

- left - It clears any floating elements on the left.

- right - It clears any floating elements on the right.

- both - It clears both left and right floats.

- none - It means that no clearing is applied.
