# Day 2 Overview

## HTML (HyperText Markup Language)
HTML is the standard markup language for creating the structure of web pages.

### Key Points

- Uses tags to define elements like headings, paragraphs, links, images, and more.
- Provides the backbone structure of a webpage.
- Elements can be nested to create complex layouts.
- HTML5 is the latest version, introducing new elements and features for modern web development.

## CSS (Cascading Style Sheets)
CSS is used for styling HTML elements, enhancing the visual presentation of web pages.

### Key Points

- Selectors target HTML elements to apply styling rules.
- Defines properties like color, font, size, layout, and animations.
- Can be included directly within HTML documents or linked externally.
- CSS frameworks like Bootstrap and Tailwind CSS offer pre-designed styles for rapid development.

## JavaScript
JavaScript adds interactivity and dynamic behavior to web pages.

### Key Points

- A versatile programming language for client-side scripting.
- Allows manipulation of HTML content, handling user interactions, and making AJAX requests.
- Can be used to create animations, validate forms, and build interactive web applications.
- Modern frameworks/libraries like React.js, Angular, and Vue.js simplify complex frontend development tasks.
