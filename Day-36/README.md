# Day 36 Overview

## JS Variables, Data types and Objects

### JS Variables

- These are the containers on which the data values are stored.

- They are of 3 types

  1. var

     - It is function scoped.

     - It can be re-declared and updated.

  2. let

     - It is block scoped.

     - It cannot be re-declared within the same scope but can be updated.

  3. const

     - It is block scoped

     - It cannot be re-declared or updated and it must be initialized at the time of declaration.

### Variable Scope

1. Global Scope - Variables declared outside any function have global scope.

2. Function Scope - Variables declared with 'var' inside a function are function-scoped.

3. Block Scope - Variable declared with 'let' or 'const' inside a block are block-scoped.

### JS Data Types

1. String - It is used to represent text.

2. Number - It is used represent both integer and floating-point number.

3. Boolean - It is used to represent logical values.

4. Undefined - It represent a variable that has been declared but not assigned a value.

5. Null - It is used to represent the intentional absence of object value.

### JS Objects 

It is a collection of key-value pairs. Each key has a value, which can be of any data type. 

Objects can be accessed by - 

1. Dot Notation.
2. Bracket Notation.
