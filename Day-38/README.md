# Day 38 Overview

## Loops in JS

These are used to execute a block of code repeatdly, either a set number of times or until a specific condition is met.

### Types of Loops

1. for Loop - It is used to repeat a block of code a specific number of times.

2. While Loop - It executes a block of code as long as the specified condition is true.

3. do..while Loop - It is similar to while loop but it ensures that the code block is executed at least once before checking the condtion.

4. for..in Loop - It is used to iterate over the properties of an object.

5. for..of Loop - It is used to iterate over the iterable objects.
