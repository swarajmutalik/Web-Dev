## CSS Grid Layout

It is a layout system in CSS that provides a flexible and efficient way to create complex, responsive and two-dimensional grid-based layouts.

### Key Concepts of CSS Grid

1. Grid Container - The parent element containing grid items is called the grid container.

2. Grid Items - The child elements within a grid container area are called grid items.

3. Grid Lines - The lines that define the boundaries between rows and columns.

4. Grid Tracks - The space between two adjacent grid lines, forming rows and columns.

5. Grid Cells - The intersection of a row and column,forming a single 'cell' within the grid.

### Grid Template Properties

1. grid-template-rows and grid-template-columns - It defines the sizes and number of rows and columns in the grid.

2. grid-template-areas - It defines name areas within the grid, allowing grid items to be placed based on these named areas.

### Grid Gap and Alignment Properties

1. gap - It is used to set the space between rows and columns.

2. justify-items and align-items - It is used to control the alignment of grid items along the horizontal and vertical axes.

3. justify-content and align-content - It is used to control the alignment of grid tracks within the grid container.

### Grid Placement Properties

1. grid-column-start, grid-column-end, grid-row-start, grid-row-end - These are used to define the start and end positions for grid items, allowing them to span multiple columns or rows.

2. grid-area - It is used to specify the area to which a grid item belongs.
