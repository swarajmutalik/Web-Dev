# Day 7 Overview

## Forms and input tags 

HTML forms employs the form element to gather input data with interactive controls.

It consists of various input types such as text, numbers, email, password, checkboxes, radio buttons etc.


## Form elements

- label - It defines labels for form elements.

- input - It is used to get input data from the form in various types.

- button - It defines a clickable button to control other elements or execute a functionality.

- select - It is used to create a dropdown list.

- textarea - It is used to get input for long text content.

- option - It is used to define options in a drop-down list.
