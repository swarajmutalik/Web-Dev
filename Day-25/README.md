## CSS Variables

- CSS Variables also known as custom properties, is a powerful feature in CSS that allows you to define reusable values that can be referenced throughout your stylesheet.

- CSS Variables are placeholders for values that can be used in multiple places within a stylesheet.

- They are defined with a specific syntax and can be used to store colors,sizes,font properties or any other CSS value.

- CSS Variables are defined using a double hypen(--) prefix and :root selector for global scope.

- Variables can be defined within specific elements or classes for local scope.

- To use a CSS variable, reference it with the var() function.

### Benefits of using CSS Variables

- Maintiainability - Variables allow you to define a value once and reuse it throughout your stylesheet making it easier to update style consistently.

- Reusability - We can define a set of variables to create a theme or design system that can be applied across multiple projects.

- Dynamic values - CSS Variables can be changed at runtime with JS, enabling dynamic styling.
