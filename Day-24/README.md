## CSS Positions

It is a set of techniques that is used in CSS that allows you to control how elements are placed on a webpage. 

### Types of CSS Postioning

1. Static - This is the default position of all the elements. 
Static elements are positioned according to the normal flow of the document.

2. Relative - In this elements are positioned according to their normal position in the document flow.

3. Absolute - In this the element is positioned relative to its nearest positioned ancestor. If no ancestor exists, then it is positioned relative to its viewport.

4. Fixed - Fixed-positioned elements are removed from the normal document flow and are positioned relative to the viewport. They remain in a fixed position even when the page is scrolled. 

5. Sticky - It is an element that behaves like a relative element until a specific srcoll position is reached, then it becomes fixed. 

6. z-index - It is used to control the stacking order of positioned eleements.
