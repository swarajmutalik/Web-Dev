# Day 35 Overview 

## Introduction to JS 

JavaScript is a high-level, interpreted programming language that is mainly used for the purpose of creating dynamic and interactive content on a website.

It is an essential component of web development,along with HTML, CSS and it is supported by all the modern web browsers. 

### Key Features of JavaScript 

- It can be used to manipulate the HTML, CSS of a webpage to create interactive effects.

- It is executed in the user's browser, allowing for real-time updates and interactions without the need to communicate with the server.

- It can be easily integrated with other technologies to enhance the functionality and appearance of a website.

- It can respond to user actions like clicking, key presses and mouse movement, making it ideal for creating responsive web apps.
