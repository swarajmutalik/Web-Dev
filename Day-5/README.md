# Day 5 Overview

## Image tag

It is an HTML tag that is used to insert images into a webpage.

## Break tag

It is an HTML tag that is used for line break. It is a self-closing element meaning it does not require a closing tag.

## Table tags

1. **table tag** - It defines the structure for organizing the data in rows and columns within a web page.

2. **tr tag** - It represents a row within an HTML table, containing individual data.

3. **th tag** - It shows a table header that typically holds the heading.

4. **td tag** - It represents a standard data cell holding some content.

5. **caption tag** - it provides a title for the entire table.

6. **thead tag** - It defines the header section of a table.

7. **tbody tag** - It represents the main content area of a table, seperating it from the header or footer.

8. **tfoot tag** - specifies the footer section of a table.

## HTML lists

It is a record related information used to display the data or any information on web pages in the ordered or unordered form.

Here is a list of all the list tags -

1. ul - defines an unordered list.

2. ol - defines an ordered list.

3. li - defines a list item
