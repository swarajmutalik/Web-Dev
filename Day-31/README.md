## CSS Transitions

It is a CSS property that provides a way to smoothly animate property changes on elements over a specified duration.

They offer a simple and efficient method to enhance user experience by adding subtle animations to elements such as changing of colors, size or positions.

The 'transition property is used to define the transition effect,specifying the properties to transition, the duration of the transition, the timing function and any delay before the transition starts. 

- property - It is used to specify the property to be transitioned.

- duration - specifies the duration of the transition effect in seconds or milliseconds. 

- timing-function - It specifies the timing function to use for the transitions, controlling the pace of the animation.

- delay - It is used to specify the delay before the transition starts.
