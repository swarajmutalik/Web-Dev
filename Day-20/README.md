## CSS Display Property

The CSS display property is used to how an element is displayed in the layout of a webpage. 

It is used to determine the type of box that an elements generates, influencing it behaviour, layout and interaction with other elements.

Some of the most common values of the display property are as follows - 

- block - The element is used to generate a block-level box, occupying the full width available and starting on a new line 

- inline - The element generates an inline-level box,allowing it to flow within the text content of a line. 

- inline-block - It is used to combine both the features of inline and block. It is useful for those elements that need to be displayed inline but require block-level properties.

- none - In this the element is removed from the document flow and it is not rendered. It is used to effectively hide the elements.
