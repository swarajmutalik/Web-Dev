# Day 43 Overview

## More about DOM

### Accessing elements by ID

1.getElementById - It returns the element with the specified ID. 

### Accessing elements by Class Name

1. getElementByClassName - It returns a live HTML collection of all the elements with the specified class names.

### Accessing elements using CSS selectors 

1. querySelector - It returns the first element within the document that matches the specified CSS selector.

2. querySelectorAll - It returns a static NodeList of all the elements within the docuement that matches the specified CSS selectors.
