# Day 37 Overview

## JS If-else Conditionals

The if-else condtionals in JS are used to execute code based on a condition. It allows you to run a code if a certain condition is meet.

### If statement

It is used to check a condition and execute a block of code if the condition evalutes to true.

### If..else statement

It is used for checking some condition and execute the block of code if it is true and if the condition is false then it returns another block of code.

### Nested if..else statements

It is used to handle complex conditions by nesting if..else block into another if..else block.

### Ternary Operator

It is used to provide a shorthand of writing if..else statements.

It takes three operands - a condition, an expression to execute if the statement is true, and an expression to execute if the condition is false.

condition ? expressionIfTrue : expressionIfFalse
