# Day 13 Overview

## HTML Entities 

These are special codes or sequences that are used to represent characters that have a special meaning in HTML, or characters that might not be easily typed on a keyboard.

They allow you to include characters that might otherwise be misinterpreted by the browser, ensuring that they display correctly.

### Why Use HTML Entities?

HTML entities are used when:

- Including reserved characters: Some characters have special roles in HTML, such as <, >, and &. These characters must be represented with entities to avoid breaking the HTML structure.

- Displaying invisible characters: Characters like non-breaking spaces need entities to prevent them from being lost or collapsed.

- Incorporating non-ASCII characters: To include characters from different languages or symbols, entities offer a way to represent them.

### HTML Code tag 

It is a tag that is used to define a piece of code or a snippet of code in a document. 

By using the code tag, you can semantically indicate that a section of text represents code, which has specific styling.

### Characteristics of code tag

- Semantic meaning: It tells the browser and other tools that the content is code, allowing for appropriate styling and handling.

- Monospace Font: It is generally displayed in a monospace or fixed-width font, helping to distinguish it from the normal text.

- Use in combination: It is often used with other tags.
