# Day 40 Overview

## Introduction to Strings

Strings are a fundamental data type in JS that is used to represent text. They are a sequence of characters enclosed in quotes.

### String Methods

1. CharAt() - returns the character at a specified index.

2. concat() - It is used to concatenate two or more strings and returns a new string.

3. includes() - It is used to determine whether one string contains another string, returning true or false.

4. indexOf() - It is used to return the index of the first occurance of a specified value in a string. It returns -1 if the value is not found.

5. lastIndexOf() - It is used to return the last occurance of a specified value in a string. It returns -1 if the value is not found.

6. Slice() - It is used to extract a part of the string and returns it as a new string, without modifying the original string.

7. toUpperCase() - It is used to convert a string to uppercase letters.

8. toLowerCase() - It is used to convert a string to lowercase letters.

9. trim() - It is used to remove the whitespaces from both the ends of a string.

10. replace() - It is used to replace a specified value with another value in a string.

11. startswith() - It is used to determine whether a string begins with the characters of a specified string.

12. endsWith() - It is used to determine whether a string ends with the characters of a speicifed string.

### String Template Literals

It allows for easier string interpolation and multi-line strings. They are enclosed in backticks(``) and can contain placeholders indicated by ${expression}.
