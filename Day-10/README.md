# Day 10 Overview

## Audio, Video and Media tags in HTML

### Audio tag

The audio tag allows you to embed audio files into a webpage, enabling sound or music to be played directly within the HTML document. It supports multiple formats, including MP3, WAV, and OGG.

Some common attributes used in audio tag are as follows -

- src : Specifies the source of the audio file.
- controls : If present, displays audio controls like play, pause, and volume.
- autoplay : If set, the audio plays automatically when the page is loaded.
- loop : Repeats the audio file continuously.
- muted : Mutes the audio by default.

### Video tag

The video tag is similar to audio but is used to embed video content. It supports a variety of formats, such as MP4, WebM, and OGG.

Some common attributes used in video tag are as follows -

- src : Specifies the source of the video file.
- controls : Displays video controls like play, pause, and full-screen.
- autoplay : Automatically plays the video when the page is loaded.
  loop : Repeats the video.
- muted : Mutes the video by default.
- poster : A placeholder image displayed before the video plays.

### SVG tag

The svg tag allows you to embed Scalable Vector Graphics, which can contain shapes, text, and images. SVG content is resolution-independent and can be styled with CSS or manipulated with JavaScript.

### img tag

The img tag is used to display images on a webpage. It supports multiple formats, including JPEG, PNG, GIF, and SVG.

Some common attributes used in img tag are as follows -

- src : Specifies the source of the image.
- alt : Provides alternative text for accessibility and search engines.
- width and height : Specify the image dimensions.
- title : Gives additional information when hovering over the image.

### iframe tag

The iframe tag is used to embed external content within a webpage, such as another website, a YouTube video, or a Google Map. It creates a small window where the external content is displayed.

Some common attributes used in iframe tag are as follows -

- src : The URL of the external content.
- width and height : The dimensions of the iframe.
- frameborder : Specifies the border style for the iframe.
- allowfullscreen : If set, allows the iframe to go full-screen.
