# Day 41 Overview

## Introduction to Arrays

Arrays in JS are used to store multiple values in a single variable.

They are list-like objects and come with a variety of built-in methods and properties to manage and manipulate the collection of data.

They can be created using the array literal syntax.

Array elements are accessed using their index, starting from 0.

### Array Methods

1. push() - It is used to add more elements to the end of the array and returns the new length of the array.

2. pop() - It is used to remove the last element from an array and returns that element.

3. unshift() - It is used to add one or more elements to the beginning of an array and returns the new length of the array.

4. shift() - It is used to remove the first element from an array and returns that element.

5. slice() - It returns a shallow copy of a portion of an array into a new array object.

6. splice() - It is used to change the contents of the array by removing, replacing or adding elements.

7. concat() - It is used to merge two or more arrays and returns a new array.

8. join() - It joins all the elements of the array into a string and then return this string.

9. reverse() - It is used to reverse the order of the elements in an array.

10. sort() - It is used to sort the elements of an array and return the array.

11. indexOf() - It returns the first index at which a given element can be found in the array, and it returns -1 if the element is not present.

12. includes() - It is used to determine whether an array contains a specified element, returning true or false as appropriate.

13. map() - It is used to create a new array populated with the results of calling a provided function on every element in the calling array.

14. filter() - It is used to create a new array with all the elements that pass the test implemented by the provided function.

15. reduce() - It executes a reducer function on each elements of the array, resulting in a single output value.
