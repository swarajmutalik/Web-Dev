# Day 26 Overview

## CSS Media Queries

CSS Media Queries is a powerful feature in CSS that allows developers to apply different styles based on certain conditions, typically related to the characteristic of the user's device or environment.

Media Queries are CSS rules that apply styles based on specific conditions, such as screen size, device orientation or resolution.

### Common Conditions for Media Queries

1. min-width / max-width - These conditions apply styles based on the minimum or maximum width of the viewport.

2. min-height / max-height - Similar to min-width and max-width, these conditions apply styles based on the height of the viewport.

3. orientation - Applies styles based on the device's orientation.

4. resolution - Conditions based on the screen's resolutions.

5. hover and pointer - These conditions check whether a device has hover capabilities or the type of pointer.
