## Id and Class Attributes in HTML

### Id Attribute 

- The Id Attribute is a unique identifier that is used to specify the document.

- It is used in CSS and JS to perform a certain task for a unique element.

- In CSS, the Id attribute is written using the # symbol followed by id.

### Class Attribute

- The class attribute is used to specify one or more class names for an HTML element. The class attribute can be used on any HTML element.

- The class name can be used by CSS and JS to perform certain tasks with the specified class name.

- In CSS, the class attribute is writtem using the "." symbol followed by class.
