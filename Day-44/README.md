# Day 44 Overview

## Inserting and Removing elements from JS

1. innerHTML - It is used to return HTML content inside an element.

2. outerHTML - It is used to return the content of the element, including the element itself.

3. tagName - It is used to return the tag name of the element in uppercase.

4. textContent - It returns or sets the text content of the element and its descendents.

5. hasAttribute - It is used to return a boolean indicating whether the specified attibute exists.

6. getAttribute - It is used to return the value of the speciifed attribute.

7. setAttribute - It is used to set the value of the specified attribute.

8. removeAttribute - It is used to remove the specififed attibute from the element.
