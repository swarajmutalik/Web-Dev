## CSS List-Style Properties

It is the primary property that is used for controlling list styling.

Some of the key components of this property are as follows -

1. list-style-type

   - It defines the type of marker or bullet used in a list.

   - Common values for unordered lists are disc,circle,square.

   - Common values for ordered lists are decimal, lower-alpha,lower-roman.

2. list-style position 

    - It is used to control the marker that is placed inside or outside the content area of the list item.

    - inside - It affects the text content that is inside the content area.

    - outside - It keeps the text aligned that is outside the content area.
