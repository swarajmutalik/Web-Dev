## SEO and Core Web Vitals in HTML

## SEO 

Search Engine Optimization is the process of improving the visibility of a website or a webpage on a search engine results page so as to make a company's website more discoverable.

## Core Web Vitals 

These are a set of metrics focused on user experience. The Core Web Vitals is an initiative to provide unified guidance from google that points to quality signals essentials to delivering a great user experience on the web.

It mainly consists of 3 components - 

1. **LCP - Largest Contentful Paint**

- It measures the timiing for a browser to render the largest text or image block in the viewport.

- LCP should occur within 2.5 seconds of when the first page begins loading to provide good user experience.

2. **FID - First Input Delay**

- This metric helps measure your user's first impression of your site's interactivity and responsiveness. 

- It looks at how fast your page loads combined with how quickly a user can interact with the page.

3. **CLS - Cumulative Layout Shift** 

- Stability is a measure by CLS, which looks at how a page moves around when loading.

- To provide good user experience, pages should maintain a CLS of less than 0.1.

## Lighthouse Report 

It is a free, automated tool for gaining valuable insights into your web pages performance and overall experience.
