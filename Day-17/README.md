# Day 17 Overview

## CSS Fonts, Text and Color Properties

### Font Properties

- font-family - It speicifies the typeface or a list that can be used for text.

- font-size - It is used to set the size of the font.

- font-weight - It is used to define the thickness or boldness of the font.

- font-style - It is used to determine whether the text can be normal, italic or oblique.

- line-height - It is used to define the space between lines of text.

### Text Properties 

- text-align - It is used to control text alignment within its container.

- text-decoration - It is used to apply decorations to the text.

- letter-spacing - It is used to define the spacing between letters in a word.

- text-transform - It is used to change the case of text.

### Color Properties 

- color - It is used to specify the text color.

- background-color - It is used to define the background color of an element.

- opacity - It is used to set the opacity of an element. A value of 0 is transparent and 1 is fully opaque.
