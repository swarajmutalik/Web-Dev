## Introduction to CSS 

CSS or Cascading style sheets, is a style language that is used to define the presentation and design of HTML documents. 

It allows developers to contol the look and feel of web pages, seperating content from presentation.

### Key features of CSS

- Seperation of Content and Presentation - By keeping styling in CSS, HTML remains focused on content structure, making it easier to manage and change styles without altering the content.

- Responsive design - CSS can adapt the layout and style of web pages based on the device or screen size, enabling a single codebase to work across multiple platforms.

- Reuseable - By externalizing styles into CSS files,you can reuse the same styles across multiple pages, reducing redundancy and improve maintainability 

- Selector Based - CSS uses selectors to target HTML elements and apply styles. This flexibility allows you to style specific elements, classes or even complex combinations.
