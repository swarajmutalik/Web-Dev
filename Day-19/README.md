## CSS Size Units

CSS size units are used to specify the dimensions, spacing and other properties related to the size of element on a web page.

There are 2 different types of units that are present.

1. Absolute Units

    They are used to represent fixed measurements that are not affected by other elements or the viewport size.

    - Pixels - It is a fixed-size unit that is commonly used in web design.

    - Inches - It is mainly used for print related designs but can  be useful in specific contexts.

2. Relative Units

    It is based on other elements or context providing flexibility and adaptibility.

    - Em - It represents the font size of the parent elements. It is scalable and relative to the surrounding context.

    - Rem - It is similar to em, but relative to the root element's font size. It provides more consistency across the document.

    - Viewport Width -
  It represents a percentage of viewport's width.

    - Viewport height - It represents a percentage of the viewport's height.

    - Viewport Minimum and Viewport Maximum - vmin is a percentage of the smaller dimensions and vmax is a percentage of the larger dimension.
