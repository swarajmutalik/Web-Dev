# Day 15 Overview 

## Selectors in CSS

### Element Selector

It targets all the elements of a specific type.

### Class Selector 

It targets all the elements with a specific class attribute. Generally the dot is used to indicate a class.

### ID Selector 

It targets an element with a specific ID attribute. Generally the hash is used to indicate an ID.


### Child Selector 

It targets the direct children of a specific element.

### Descendant Selector 

It targets the elements that are descendants of a specific element.


### Universal Selector 

It targets all the elements that are present in the document.

### Pseudo Selector 

It targets the specific parts of an element.
