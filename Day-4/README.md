# Introduction to HTML Tags

## Heading tags 

The HTML heading tag is used to define the headings of a page. HTML provides six levels of heading elements, ranging from h1(highest level) to h6 (lowest level).

### Why are the Heading so Important ?

1. SEO Impact 
2. Highlighting key topics.

## Paragraph tags 

The HTML tag is used to define a paragraph. It is has a opening and a closing tag. 

## Link tag 

The HTML tag links the documents with an external resource. The link tag is mainly used to link external style sheets.
