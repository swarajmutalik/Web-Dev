# Day 32 Overview

## CSS Animation

CSS animations allow you to create dynamic and visually engaging effects on the webpages by animating the properties of HTML elements over a specified duration.

CSS Animations are defined using '@keyframes', which is used to define the intermediate stpes of keyframes of the animation.

Each keyframe is used to define the values of the CSS properties at a specific point in time during the animation.

The 'animation' property is then used to apply the animation to a spcific element, in which different parameters are defined.

### Points to keep in mind

1. @keyframe Rule - It is used to define the intermediate steps of the animation using percentage values or specific time intervals.

2. Animation Property - It is applied to the animation of an element, and in this we can specify the keyframe, duration, timing function.delay, iteration count, and other parameters.
