# CSS Transform

- The transform property in CSS allows you to visually manipulate elements by applying transformations such as translation, rotation, scaling, and skewing.

- Some of the transform functions are as follows -

  - `translate(x, y)`

    - Moves an element from its current position.
    - x: Horizontal translation (e.g., 50px, 10%).
    - y: Vertical translation (optional, default is 0).

  - `rotate(angle)`

    - Rotates an element clockwise by the specified angle.
    - angle: Rotation in degrees (deg), radians (rad), gradians (grad), or turns (turn).

  - `scale(x, y)`

    - Scales the size of an element.
    - x: Horizontal scaling factor.
    - y: Vertical scaling factor (optional, default is x).

  - `skew(x-angle, y-angle)`
    - Skews an element along the X and Y axes.
    - x-angle: Skew angle along the X-axis.
    - y-angle: Skew angle along the Y-axis (optional, default is 0).
