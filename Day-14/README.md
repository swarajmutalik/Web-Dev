# Day 14 Overview 

## Inline, Internal and External CSS

### Inline CSS 

This type of styling is used directly within the HTML tags using the 'style' attribute. It is less maintainable but useful for specific styles.

### Internal CSS 

In this type of styling, the CSS is written in the 'style' tag in an HTML document. It helps style specific pages without affecting others.

### External CSS

Linked from an external file, typically with a '.css' extension. This is the most maintainable and scalable method, as it allows you to apply styles across multiple pages or entire websites.
