# Day 45 Overview 

## Events, Bubbling, setInterval and setTimeout

An event is an interaction from the user or the browser that can be detecd and hanlded by JS.

### Event Handling 

JS allows the developers to specify what should happen when a particular event occuers.

### Event Bubbling 

It is a concept in the event propagation mechanism of the DOM. When an event occurs in an element inside another element, both the elements will receive the event. The event starts from the target element and bubbles up to the outer elements. 

### setInterval and setTimeout 

setInterval is a JS function that repeatedly calls a function or executes a code snippet with a fixed time delay between each call.

setTimeout is a JS function that sets a timer which executes a function or a specified piece of code once the timer expires.
