## CSS Transforms

It is a powerful feature in CSS that allows you to modify the apperance and layout of elements in your webpage by applying various transformations.

CSS Transformations do not affect the actual layout of the document, instead they alter the visual representation of elements on the screen.

### Basic Properties

1. Translate - It is used to move an element horizontally or vertically.

2. Scale - It is used to resize an element by specifying scale factors for both the horizontal and vertical dimensions. 

3. Rotate - It is used to rotate an element clockwise or anticlockwise around a specified point. 

4. Skew - It distorts an element by skewing it along the horizontal or vertical axes.

5. Transform-origin - It allows you to specify the point around which a transformation should be applied.
